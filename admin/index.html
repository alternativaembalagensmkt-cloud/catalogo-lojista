Painel do Catálogo

Pra editar o catálogo você precisa de um token de acesso do GitHub. Ele fica salvo só no seu navegador — ninguém mais vê.

  1. Acesse github.com/settings/tokens?type=beta e entre com sua conta
  2. Clique em Generate new token
  3. Em Repository access, escolha Only select repositories e selecione o repositório do catálogo
  4. Em Permissions → Repository permissions, ache Contents e mude pra Read and write
  5. Clique em Generate token e cole aqui embaixo
